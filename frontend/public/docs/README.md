# Welcome to Stellar x402

Stellar-native implementation of the x402 HTTP payment protocol - enabling seamless micropayments for API access powered by the Stellar blockchain.

## What is x402?

x402 is an open protocol that extends HTTP with native payment capabilities, allowing APIs to monetize access through micropayments without complex payment integrations. This implementation brings that power to the Stellar ecosystem.

## Why Stellar x402?

- **Fast & Low-Cost**: Leverage Stellar's 3-5 second settlement times and minimal fees
- **USDC Native**: Built-in support for USDC on Stellar
- **Developer-Friendly**: Drop-in axios replacement for clients, Express middleware for servers
- **Production Ready**: Battle-tested payment verification and settlement
